# CryptoPulse 🚀

Aplicación de seguimiento de criptomonedas desarrollada en Flutter. Este proyecto es un espacio de aprendizaje enfocado en la construcción de aplicaciones móviles robustas y escalables.

## ⚙️ Configuración del proyecto

Para ejecutar este proyecto, necesitas configurar tus propias variables de entorno:

1. Copia el archivo `.env_template` y renómbralo a `.env`.
2. Abre el archivo `.env` y coloca tu API Key correspondiente:
   `API_KEY=tu_api_key_aqui`
3. Asegúrate de que el archivo `.env` esté en la raíz del proyecto. **No compartas este archivo.**

## 💻 Entorno de Desarrollo (VS Code)

Este repositorio incluye la configuración de `.vscode` para garantizar una experiencia de desarrollo estandarizada:

- **Formato Automático:** El proyecto está configurado para ordenar los _imports_ y formatear el código automáticamente al guardar (`settings.json`).
- **Ejecución Directa:** Incluye `launch.json` para que puedas compilar y ejecutar la aplicación inmediatamente con `F5`.

## 🛠 Tecnologías

- Flutter & Dart
- Riverpod (Gestión de estado)
- GoRouter (Navegación)
- Clean Architecture

## 🎯 Objetivo

Este es un proyecto personal donde aplico principios SOLID y arquitectura limpia para entender cómo estructurar aplicaciones de nivel profesional.
